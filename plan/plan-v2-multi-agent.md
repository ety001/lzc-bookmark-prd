```
│ L2 - Phase 协调层 │ │ L2 - Phase 协调层 │ │ L2 - Phase 协调层 │
│ 服务端 Phase     │ │ 插件 Phase       │ │ Web 端 Phase     │
│ Backend Agent   │ │ Extension Agent │ │ Web Agent       │
└─────────────────┘ └─────────────────┘ └─────────────────┘
        │                   │                   │
        ▼                   ▼                   ▼
┌─────────────────────────────────────────────────────────────┐
│ L3 - 任务执行层 (Task Execution Layer)                       │
│ 具体开发任务、单元测试、代码提交                             │
│ 多个 Worker Agent 并行执行                                   │
└─────────────────────────────────────────────────────────────┘
```

### 分层说明

| 层级 | 职责 | 决策粒度 | 更新频率 |
|------|------|----------|----------|
| **L1 里程碑层** | 整体进度、资源调配、风险管控 | Phase 级别 | 每日 |
| **L2 Phase 协调层** | Phase 内任务分解、依赖管理、质量把控 | 任务组级别 | 每 4 小时 |
| **L3 任务执行层** | 具体编码、测试、文档编写 | 单任务级别 | 实时 |

3. **延后执行** 低重要性 + 低紧急性任务（缓冲时间）

## 6. 并行任务调度

### 6.1 调度算法

```python
# 伪代码：多 Agent 任务调度
def schedule_tasks(tasks, agents):
    """
    基于依赖关系和 Agent 技能的任务调度
    """
    schedule = []
    ready_queue = [t for t in tasks if not t.dependencies]
    
    while ready_queue or tasks:
        # 为每个空闲 Agent 分配任务
        for agent in agents:
            if agent.is_idle and ready_queue:
                # 选择优先级最高的可执行任务
                task = select_task(ready_queue, agent.skills)
                agent.assign(task)
                ready_queue.remove(task)
        
        # 等待当前批次任务完成
        wait_for_completion()
        
        # 更新就绪队列（依赖已满足的任务）
        completed_tasks = get_completed_tasks()
        for task in tasks:
            if all(dep in completed_tasks for dep in task.dependencies):
                ready_queue.append(task)
                tasks.remove(task)
    
    return schedule
```

### 6.3 资源冲突解决

| 冲突类型 | 场景 | 解决策略 |
|----------|------|----------|
| **Agent 冲突** | 多个任务需要同一个 Agent | 优先级队列，高优先级先执行 |
| **依赖冲突** | 任务 A 等待任务 B，B 被阻塞 | Coordinator 介入，调整资源 |
| **环境冲突** | 多个任务需要同一测试环境 | 时间片轮转，或并行环境 |
| **代码冲突** | 多个 Agent 修改同一文件 | 代码所有权划分 + 频繁合并 |

### 7.1 依赖类型

| 依赖类型 | 说明 | 示例 | 管理方式 |
